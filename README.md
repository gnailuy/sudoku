# Sudoku

A simple Sudoku game.

## Build

```bash
go build
```

## Play

### Play with a random board

```bash
./sudoku
./sudoku -l easy
```

### Play with a custom board

```bash
./sudoku -i .56.4.7...1.5....6.......19...9.....3.58..2...4...6...1.....93....4....22.3.1....
```

### Show help

```bash
./sudoku -h
```
