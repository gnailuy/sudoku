# Sudoku

A simple Sudoku game.

## Build

```bash
go build
```

## Play

### Play with a random board

```bash
./sudoku
./sudoku -l easy
```

### Play with a custom board

```bash
./sudoku -i ...61...9.8...........38..74...69.382...8...6..17........3.4......5.1683.9..7.5..
```

### Show help

```bash
./sudoku -h
```
